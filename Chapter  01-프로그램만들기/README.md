# Chapter 01 - 프로그램 만들기

## 📖 학습 내용
- Visual Studio 프로젝트 템플릿 만들기
- 소스 파일 생성하기
- 전처리기 정의에 `_CRT_SECURE_NO_WARNINGS` 추가하기
- 프로젝트 템플릿 내보내기

## 💡 핵심 개념

### 전처리기 설정
프로젝트 속성 → C/C++ → 전처리기 → 전처리기 정의에 아래 추가:
_CRT_SECURE_NO_WARNINGS

> scanf 등 보안 경고를 없애주는 설정이에요!

### 템플릿 내보내기
프로젝트 설정 완료 후 매번 반복 작업을 줄이기 위해 템플릿으로 저장:
프로젝트 → 템플릿 내보내기 → 프로젝트 템플릿 → 완료

> 다음 프로젝트부터 동일한 설정을 자동으로 적용할 수 있어요!
